# Better-Archery
This is a port and, so to speak, a “fork”, since the source code for the Better Archery mod,
created by Zaggy1024, is not available. This source code is in the ultra-beta phase and, for the time being, isn’t well-written, since I had to decompile it and create my own, furthermore, the code contains comments in Spanish that serve as a guide for me, and little by little I’ll add the features that the mod had.

The tools used to decompile the `.class` files were **cfr-0.152** and **JD-GUI**. This means that this is a recreation, since I do not have access to the original source code

Original mod: [https://www.curseforge.com/minecraft/mc-mods/better-bows](https://www.curseforge.com/minecraft/mc-mods/better-bows)

**NOTE**: It looks like this is going to take a while, it’s taking me a long time to recreate the mod. I can’t even imagine how difficult it will be to adapt it to versions later than 1.13, it’s very complicated to understand functions like this one, `field_77779_bT`, and I still have other mods in mind that I want to adapt

Currently, this port is 96% complete. For now, optimization won't be my top priority, I'll port it first and then work on optimizations and other tasks, including bug fixes
